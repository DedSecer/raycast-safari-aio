# Safari-Aio

Search Safari tabs, bookmarks, and history in a single Raycast command.

## Features

- Unified search for Safari tabs, bookmarks, and browsing history.
- Each result is labeled as Tab, Bookmark, or History.
- Configurable source priority with 3 levels:
	- Priority 1
	- Priority 2
	- Priority 3
- Configurable list display mode:
	- Flat mixed list
	- Grouped sections
- Optional inclusion of iCloud remote tabs from other devices.
- Fuzzy search toggle.

## Preferences

In Raycast command preferences for Search Safari-Aio:

- Local Safari Browser: Safari or Safari Technology Preview
- Devices: include iCloud tabs
- List Mode: flat or section
- Priority 1 / 2 / 3: choose source order among Tabs, Bookmarks, and History
- Fuzzy Search: enable or disable fuzzy matching
- Max Results Per Source: 50 / 100 / 200

## Notes

- This extension is macOS-only.
- Safari history and bookmarks may require Full Disk Access for Raycast.
